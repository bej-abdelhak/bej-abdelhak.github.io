نبذة

قائد تقني ومطوّر Full-Stack بخبرة تتجاوز 16 عاماً في بناء برمجيات إنتاجية متكاملة من طرف إلى طرف — قاعدة البيانات، والـ API، والويب، والموبايل، والواجهة نفسها. أقود قسم التقنية في مجموعة أعمال متعددة القطاعات، حيث أمتلك معمارية وتسليم منصة مؤسسية لإدارة المعارض والفعاليات — سبع بوابات مبنية على Laravel API وبوابات React وتطبيق Expo/React Native، تعمل في الإنتاج للتجّار والمستثمرين والرعاة في المملكة العربية السعودية. دخلتُ هندسة البرمجيات عبر الإخراج الفني وتصميم UX/UI، وما زال ذلك يشكّل كل ما أبنيه. أعمل بالذكاء الاصطناعي كجزء من أدواتي لا كتجربة — Claude Code وCursor وGitHub Copilot في دورتي اليومية، وأصمّم مسارات الوكلاء وبوابات المراجعة التي تُبقي مخرجاتها بمستوى الإنتاج.
